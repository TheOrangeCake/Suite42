Bienvenue sur le site Transcendance. La protection de vos données personnelles est une priorité. Cette politique de confidentialité explique quelles données nous collectons, comment nous les utilisons, et quels sont vos droits conformément au Règlement Général sur la Protection des Données (RGPD).

## 1. Responsable du traitement

## 2. Données collectées

Nous collectons les catégories de données suivantes :

### a) Données issues de l’API 42
Dans le cadre de l’utilisation de l’API de 42, nous pouvons collecter :
- Identifiant utilisateur
- Nom / pseudonyme
- Adresse email (si accessible)
- Informations de profil (niveau, projets, cursus, etc.)

Ces données sont récupérées uniquement avec votre autorisation via le système d’authentification de 42 (OAuth).

### b) Données fournies via la chatbox
Lorsque vous utilisez notre chatbox :
- Messages échangés
- Données techniques (adresse IP, navigateur, horodatage)

### c) Données techniques
- Adresse IP
- Type de navigateur
- Cookies et données de session

---

## 3. Finalités du traitement

Vos données sont utilisées pour :
- Fournir et améliorer les fonctionnalités du site
- Permettre l’authentification via l’API 42
- Assurer le fonctionnement de la chatbox
- Répondre à vos demandes via le chat
- Garantir la sécurité du site
- Respecter nos obligations légales

---

## 4. Base légale

Les traitements reposent sur :
- Votre consentement (chatbox, cookies, API 42)
- L’exécution d’un service (fonctionnement du site)
- L’intérêt légitime (sécurité, amélioration)

---

## 5. Conservation des données

Les données sont conservées :
- Pendant la durée d’utilisation du service
- Maximum [durée à définir, ex : 12 mois] pour les logs
- Les messages de chat peuvent être conservés pour améliorer le service, puis anonymisés

---

## 6. Partage des données

Vos données ne sont pas vendues.

Elles peuvent être partagées uniquement avec :
- Prestataires techniques (hébergement, chatbox)
- API 42 (dans le cadre de l’authentification)
- Autorités légales si requis

Tous les prestataires respectent le RGPD.

---

## 7. Cookies

Nous utilisons des cookies pour :
- Assurer le bon fonctionnement du site
- Améliorer l’expérience utilisateur

Un bandeau de consentement vous permet de gérer vos préférences.

Nous mettons en œuvre des mesures de sécurité adaptées :
- Chiffrement des communications (HTTPS)
- Accès restreint aux données
- Surveillance et protection contre les accès non autorisés
